A compatch for more religious tenets and various religion mods allowing you to create a custom religion using many more tenets. 

Mods Patched include: 

Elf Destiny
Regula Magistri
Carnalitas Dei 
Sapphos Daughter. 



ONLY Tenets have been patched. 

you require the mods you wish to use with this patch, if you do not wish to use one of the above mentioned mods either "#" out its entire section or delete the section (they are named in the file) 
Reccommend you place this as low in your load order as possible (will NEED to be below ALL religion mods)
